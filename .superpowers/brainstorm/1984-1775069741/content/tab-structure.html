タブのカテゴリ分けはどうする？

team-f-cc-lab では All / Plugin / Skill / Guide のタブがある。あなたのポートフォリオでは？

A

All / Lesson / Works

「Lesson」で学習課題（lesson1, 2, 3...）を、「Works」で今後の自主制作物を分類。成長の過程と成果を分けて見せる。

B

All / Frontend / App / Other

技術カテゴリで分類。フロントエンド系・アプリ系・その他。エンジニアとしてのスキル領域を軸に見せる。

C

タブなし（シンプル一覧）

まだ作品が少ないのでタブ不要。カード一覧だけでスッキリ見せて、増えたら後からタブを追加。
